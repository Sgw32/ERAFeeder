# ERAFeeder

Программа ERAFeeder предназначена для эмуляции команд джойстика в стендовом режиме работы конвертоплана.

Для работы программы требуется установленное ПО vJoy.

## Порядок работы с программой:

После успешного запуска отобразится окно управления программой. Следует настроить при необходимости все регулируемые элементы, такие как уровень газа, и постоянные времени, а также настроить отключение управления по рысканию.
После соединения с аппаратом в Mission Planner следует задать управление по джойстику.

 - Ось X - тангаж
 - Ось Y - крен
 - Ось Z - газ
 - Ось Rx - рыскание
 - Ось Rz - конверсия
 - Кнопка "0" - "Arm"

После этого, расположив аппарат на стенде в горизонтальном положении, нажать кнопку "СТАРТ". 
Двигатели запустятся и выйдут на заданный уровень мощности. Этап отображается индикацией "ПУСК", после чего программа перейдёт в режим "БЕЗДЕЙСТВИЕ".  После этого через равные интервалы времени программа генерирует новые желаемые углы отклонения джойстика по каналам тангажа, крена и рыскания, а также и по газу. В случае выпадения числа с вероятностью 1:10 аппарат совершает конверсию из коптера в самолёт, и из самолёта в коптер, после чего снова переходит в режим "БЕЗДЕЙСТВИЕ". 
Регулировка постоянных времени и уровня газа может осуществляться "на ходу". Рыскание также можно отключить при работающем аппарате.
После проведения всех требуемых испытаний с помощью программы следует нажать кнопку "СНЯТЬ СО СТЕНДА", после чего все оси переведутся в среднее состояние, конверсия будет переведена в коптер. Отобразится статус "СТОП ГОТОВ". После того, как аппарат будет надёжно закреплен на стенде, следует нажать кнопку "СТОП", после чего газ сбросится до нуля. При этом аппарат будет все также в режиме "Armed". Следует через Mission Planner нажать кнопку Disarm вручную.
